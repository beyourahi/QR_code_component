### Tech stack used:

-   [Astro](https://astro.build/)
-   [Typescript](https://www.typescriptlang.org/)
-   [Tailwind CSS](https://tailwindcss.com/)
-   [Vercel](https://vercel.com/)

<br />

### Steps to run:

1. Download [pnpm](https://pnpm.io/) package manager:

```
npm install -g pnpm
```

2. Install dependencies:

```
pnpm install
```

3. Run dev server:

```
pnpm dev
```
